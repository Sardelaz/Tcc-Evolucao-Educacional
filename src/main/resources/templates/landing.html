📊 Como Evoluir?

O teu progresso é medido por Pontos de Experiência (XP). Ganhas XP ao resolver questões e completar módulos.

  • 🔘 Ferro: 0 a 1000 XP
  • 🥉 Bronze: 1001 a 3000 XP
  • 🥈 Prata: 3001 a 6000 XP
  • 🥇 Ouro: 6001 a 10000 XP
  • 💎 Lenda: +10000 XP
🔥 Dica: Mantém a tua ofensiva diária ativa para multiplicar os teus ganhos de XP!
